# Certificação C# ALURA

Esse é uma formação realizada na plataforma da Alura, a <b>Formação C# e orientação a objetos</b> da Alura é um guia de aprendizado completo para iniciar
uma carreira na programação <b>C#</b>. Nesse curso apresenta-se os fundamentos da linguagem com as boas práticas ate como fazer uma boa
<b>modelagem orientada a objetos</b>.Conteúdos fundamentais para um posterior mergulho mais profundo no ecossistema <b>.NET</b>

## C#

O <b>C#</b> é uma linguagem elegante e <b>fortemente tipada orientada a objetos</b>.Com uma sintaxe <b>altamenta expressiva</b>, ela permite
a criação de aplicativos seguros e robustos que sao executados no ecossistema do <b>.NET</b>
